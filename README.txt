Deutsch Quiz — 개인 독일어 단어 퀴즈 웹앱

기능
- 독일어 → 한국어, 한국어 → 독일어
- 명사 성(der/die/das), 복수형 시험
- 태그를 여러 개 붙여 어근별 / 문법별 / 단원별 시험
- 여러 태그를 골라 섞어서 출제
- 오답 우선 / 오답만 시험
- 단어 개별 입력, 일괄 붙여넣기, JSON 백업·복원

아이폰 홈 화면 앱으로 쓰는 가장 쉬운 방식
1) 이 폴더 안의 파일을 무료 정적 웹 호스팅(GitHub Pages, Netlify, Cloudflare Pages 등)에 올린다.
2) 아이폰 Safari로 올라간 주소를 연다.
3) 공유 버튼 → 홈 화면에 추가를 누른다.
4) 이후 홈 화면의 Deutsch Quiz 아이콘으로 실행한다.

중요
- 단어 데이터는 브라우저/기기에 저장된다. 다른 기기와 자동 동기화되지 않는다.
- 단어를 많이 입력했다면 데이터 메뉴에서 JSON 백업을 가끔 저장해 둔다.
- 먼저 테스트하려면 index.html을 데스크톱 브라우저에서 바로 열어도 된다.
